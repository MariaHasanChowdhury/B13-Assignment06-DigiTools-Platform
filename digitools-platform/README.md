# DigiTools 🚀
DigiTools is a modern digital product marketplace built with React.js and Tailwind CSS.  
Users can explore premium digital tools, add products to cart, remove items, and proceed to checkout with a clean and responsive UI.
-----

## ✨ Features
- 🛒 Add products to cart with live cart count
- ❌ Remove selected products from cart
- 🔔 Toast notifications using React-Toastify
- 💳 Proceed to checkout functionality
- 📱 Fully responsive design
- 🎨 Modern Figma-inspired UI
- 📦 Product data loaded from JSON
---------

## ⚙️ Tech Stack
- React.js – Frontend library
- JavaScript (ES6+) – Application logic
- Tailwind CSS – Styling
- DaisyUI – UI Components
- React Toastify – Notifications
- Vite – Fast development build tool
- Vercel – Deployment
